    6. **`_voice_lead_pair(prev_midis, next_pitch_classes)`** helper (per D-06, RESEARCH section 3):
       - For each pitch class in next chord, find nearest octave placement to minimize distance from prev chord voices
       - Handle different chord sizes: if next has more notes, add extra in nearest position; if fewer, match by closest
       - Return list of MIDI ints sorted ascending
       - Algorithm:
         ```python
         def _voice_lead_pair(prev_midis, next_pitch_classes):
             result = []
             for i, pc in enumerate(next_pitch_classes):
                 if i < len(prev_midis):
                     ref = prev_midis[i]
                 else:
                     ref = sum(prev_midis) // len(prev_midis)  # center of prev chord
                 # Find nearest octave of this pitch class to reference
                 candidate = ref + ((pc - ref % 12) % 12)
                 if candidate - ref > 6:
                     candidate -= 12
                 result.append(candidate)
             return sorted(result)
         ```
