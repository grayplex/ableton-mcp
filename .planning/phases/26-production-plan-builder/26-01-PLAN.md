    Internal helper function `_build_plan_sections(sections: list[dict]) -> list[dict]`:
    1. `sections = copy.deepcopy(sections)` -- CRITICAL: deep copy to avoid mutating blueprint registry
    2. Calculate cumulative bar_start: start at bar=1, each section gets bar_start=bar, then bar += section["bars"]
    3. Build output entries for each section: `{"name": s["name"], "bar_start": s["bar_start"], "bars": s["bars"], "roles": s.get("roles", [])}`
    4. Only include `"transition_in"` key if it exists in the source section entry (per Phase 25 D-04: first section/intro has no transition_in)
    5. Return the list of output section dicts
