Rewrite handlers/notes.py completely. Import Live at module level, import _resolve_clip_slot from handlers/clips.py, import command from registry.

Implement 5 handlers in NoteHandlers mixin class:

1. `_add_notes_to_clip` (rewrite existing) -- `@command("add_notes_to_clip", write=True)`
   - Extract track_index, clip_index, notes from params
   - Use _resolve_clip_slot, check has_clip
   - Validate each note: pitch 0-127, velocity 1-127, duration > 0. Error messages include the actual out-of-range value.
   - Build Live.Clip.MidiNoteSpecification objects for each note (pitch, start_time, duration, velocity, mute)
   - Call clip.add_new_notes(tuple(note_specs)) -- append mode, not overwrite
   - Return {"note_count": len(notes), "clip_name": clip.name}

2. `_get_notes` -- `@command("get_notes")` (READ command, no write=True)
   - Use _resolve_clip_slot, check has_clip
   - Call clip.get_notes_extended(0, 128, 0.0, clip.length) to get all notes
   - Convert MidiNote objects to dicts with 5 properties: pitch, start_time, duration, velocity, mute (use bool(note.mute) for consistent JSON)
   - Sort by start_time ascending, ties broken by pitch ascending
   - Return {"note_count": len(notes), "notes": notes}

3. `_remove_notes` -- `@command("remove_notes", write=True)`
   - Extract optional params: pitch_min (default 0), pitch_max (default 127), start_time_min (default 0.0), start_time_max (default clip.length)
   - Convert min/max to from/span format: from_pitch=pitch_min, pitch_span=pitch_max - pitch_min + 1, from_time=start_time_min, time_span=start_time_max - start_time_min
   - Count notes before removal via get_notes_extended with same range
   - Call clip.remove_notes_extended(from_pitch, pitch_span, from_time, time_span)
   - Return {"removed_count": count}
   - Claude's discretion: When ALL range params are omitted (all defaults), this removes all notes in clip -- acceptable since the user may want to clear a clip entirely.

4. `_quantize_clip_notes` -- `@command("quantize_notes", write=True)`
   - Extract grid_size (default 0.25) and strength (default 1.0, range 0.0-1.0)
   - Read all notes with get_notes_extended(0, 128, 0.0, clip.length)
   - If no notes, return {"quantized_count": 0}
   - Extract note data to list of dicts
   - Quantize each note's start_time: nearest_grid = round(original / grid_size) * grid_size, new_start = original + (nearest_grid - original) * strength
   - Remove all notes with remove_notes_extended(0, 128, 0.0, clip.length)
   - Rebuild as MidiNoteSpecification objects, call add_new_notes(tuple(specs))
   - Return {"quantized_count": len(note_data)}

5. `_transpose_clip_notes` -- `@command("transpose_notes", write=True)`
   - Extract semitones (int)
   - Read all notes with get_notes_extended(0, 128, 0.0, clip.length)
   - If no notes, return {"transposed_count": 0}
   - FIRST PASS: validate ALL notes -- for each, compute new_pitch = note.pitch + semitones. If any new_pitch < 0 or > 127, raise ValueError with message including the note's current pitch, the semitones value, and the resulting out-of-range pitch.
   - SECOND PASS (only if all valid): build note_data dicts with transposed pitches
   - Remove all notes with remove_notes_extended(0, 128, 0.0, clip.length)
   - Rebuild as MidiNoteSpecification objects, call add_new_notes(tuple(specs))
   - Return {"transposed_count": len(note_data)}

All handlers follow the existing pattern: try/except with self.log_message on error, then re-raise.
