Add a private helper method and 12 new handler methods to the DeviceHandlers class in devices.py.

**Helper: `_resolve_drum_pad(self, device, note)`**
- Iterate `device.drum_pads`, find pad where `pad.note == note`
- If found, return pad
- If not found, raise ValueError with message: `f"No drum pad with note {note} found in '{device.name}'. Available notes: {[p.note for p in device.drum_pads if p.chains]}"`

**Simpler handlers (5):**

1. **`_crop_simpler`** (`@command("crop_simpler", write=True)`):
   - Use `self._resolve_device(params)` to get device
   - Guard: `if not hasattr(device, 'playback_mode'): raise ValueError(f"Device '{device.name}' is not a Simpler device")`
   - Guard: `if device.sample is None: raise ValueError("No sample loaded in Simpler")`
   - Call `device.crop()`
   - Return: `{"cropped": True, "device_name": device.name}`

2. **`_reverse_simpler`** (`@command("reverse_simpler", write=True)`):
   - Same device resolution and Simpler guard as above
   - Guard sample is not None
   - Call `device.reverse()`
   - Return: `{"reversed": True, "device_name": device.name}`

3. **`_warp_simpler`** (`@command("warp_simpler", write=True)`):
   - Params: standard device params + `mode` (str: "as", "double", "half"), `beats` (float, only for mode="as")
   - Simpler guard + sample guard
   - If mode == "as": guard `device.can_warp_as`, raise ValueError if False. Require beats param. Call `device.warp_as(beats)`.
   - If mode == "double": guard `device.can_warp_double`, raise ValueError if False. Call `device.warp_double()`.
   - If mode == "half": guard `device.can_warp_half`, raise ValueError if False. Call `device.warp_half()`.
   - Else: raise ValueError(f"Invalid warp mode '{mode}'. Use 'as', 'double', or 'half'.")
   - Return: `{"warped": True, "device_name": device.name, "mode": mode}`

4. **`_get_simpler_info`** (`@command("get_simpler_info")`):
   - Simpler guard (hasattr check on playback_mode)
   - Build result: `{"device_name": device.name, "playback_mode": device.playback_mode, "playback_mode_label": {0: "Classic", 1: "One-Shot", 2: "Slicing"}.get(device.playback_mode, "unknown"), "can_warp_as": device.can_warp_as, "can_warp_double": device.can_warp_double, "can_warp_half": device.can_warp_half}`
   - If `device.sample is not None`: add `"sample": {"file_path": device.sample.file_path, "length": device.sample.length, "sample_rate": device.sample.sample_rate, "slices": list(device.sample.slices), "slicing_sensitivity": device.sample.slicing_sensitivity, "slicing_style": device.sample.slicing_style}`
   - Else: add `"sample": None`
   - Return result

5. **`_set_simpler_playback_mode`** (`@command("set_simpler_playback_mode", write=True)`):
   - Params: standard device params + `mode` (int: 0=Classic, 1=One-Shot, 2=Slicing)
   - Simpler guard
   - Validate mode in {0, 1, 2}, raise ValueError with label mapping if not
   - Set `device.playback_mode = mode`
   - Return: `{"device_name": device.name, "playback_mode": device.playback_mode, "playback_mode_label": {0: "Classic", 1: "One-Shot", 2: "Slicing"}.get(device.playback_mode, "unknown")}`

**Simpler slice handlers (3 -- SMPL-05):**

6. **`_insert_simpler_slice`** (`@command("insert_simpler_slice", write=True)`):
   - Params: standard device params + `time` (int, sample frames)
   - Simpler guard + sample guard
   - Call `device.sample.insert_slice(time)`
   - Return: `{"inserted": True, "device_name": device.name, "time": time, "slices": list(device.sample.slices)}`

7. **`_move_simpler_slice`** (`@command("move_simpler_slice", write=True)`):
   - Params: standard device params + `source_time` (int), `destination_time` (int)
   - Simpler guard + sample guard
   - Call `device.sample.move_slice(source_time, destination_time)`
   - Return: `{"moved": True, "device_name": device.name, "source_time": source_time, "destination_time": destination_time, "slices": list(device.sample.slices)}`

8. **`_remove_simpler_slice`** (`@command("remove_simpler_slice", write=True)`):
   - Params: standard device params + `time` (int, sample frames)
   - Simpler guard + sample guard
   - Call `device.sample.remove_slice(time)`
   - Return: `{"removed": True, "device_name": device.name, "time": time, "slices": list(device.sample.slices)}`

Also add `_clear_simpler_slices` (`@command("clear_simpler_slices", write=True)`):
   - Simpler guard + sample guard
   - Call `device.sample.clear_slices()`
   - Return: `{"cleared": True, "device_name": device.name, "slices": list(device.sample.slices)}`

**DrumPad handlers (2):**

9. **`_set_drum_pad_mute`** (`@command("set_drum_pad_mute", write=True)`):
   - Params: standard device params + `note` (int, MIDI note), `mute` (bool)
   - Guard: `if not device.can_have_drum_pads: raise ValueError(f"Device '{device.name}' is not a Drum Rack")`
   - `pad = self._resolve_drum_pad(device, note)`
   - `pad.mute = mute`
   - Return: `{"note": note, "name": pad.name, "mute": pad.mute}`

10. **`_set_drum_pad_solo`** (`@command("set_drum_pad_solo", write=True)`):
    - Same drum rack guard + pad resolution
    - Params: `note` (int), `solo` (bool), `exclusive` (bool, default False)
    - If exclusive and solo: iterate all `device.drum_pads`, unsolo each (`pad.solo = False`) before soloing target
    - `pad.solo = solo`
    - Return: `{"note": note, "name": pad.name, "solo": pad.solo, "note_text": "DrumPad solo does not auto-unsolo other pads. Use exclusive=True to solo exclusively."}`

11. **`_delete_drum_pad_chains`** (`@command("delete_drum_pad_chains", write=True)`):
    - Drum rack guard + pad resolution
    - `pad.delete_all_chains()`
    - Return: `{"deleted": True, "note": note, "name": pad.name}`

**Plugin preset handlers (1):**

12. **`_list_plugin_presets`** (`@command("list_plugin_presets")`):
    - Guard: `if not hasattr(device, 'presets'): raise ValueError(f"Device '{device.name}' is not a plugin device (VST/AU). Only plugin devices have presets.")`
    - Return: `{"device_name": device.name, "presets": list(device.presets), "selected_preset_index": device.selected_preset_index, "preset_count": len(device.presets)}`

13. **`_set_plugin_preset`** (`@command("set_plugin_preset", write=True)`):
    - Same plugin guard
    - Params: standard device params + `preset_index` (int)
    - Validate: `if preset_index < 0 or preset_index >= len(device.presets): raise IndexError(...)`
    - `device.selected_preset_index = preset_index`
    - Return: `{"device_name": device.name, "selected_preset_index": device.selected_preset_index, "preset_name": device.presets[device.selected_preset_index]}`

**A/B compare handler (1):**

14. **`_compare_ab`** (`@command("compare_ab", write=True)`):
    - Guard: `if not device.can_compare_ab: raise ValueError(f"Device '{device.name}' does not support A/B comparison")`
    - Params: standard device params + `action` (str: "save" or "toggle")
    - If action == "save": call `device.save_preset_to_compare_ab_slot()`, return with action="saved"
    - If action == "toggle": currently no toggle function in LOM -- just return current state
    - Actually, the LOM only provides `save_preset_to_compare_ab_slot()` and read-only `is_using_compare_preset_b`. So simplify:
    - `action` param optional. If provided and == "save": call `device.save_preset_to_compare_ab_slot()`
    - Return: `{"device_name": device.name, "can_compare_ab": device.can_compare_ab, "is_using_compare_preset_b": device.is_using_compare_preset_b, "action": action or "info"}`

That is 14 handler methods total (plus the helper). But to align with 12 "capability tools" at the MCP level, we can group the slice operations under fewer MCP tools. At the handler level, keep them separate for clarity.
