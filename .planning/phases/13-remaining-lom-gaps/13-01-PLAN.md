**scenes.py -- Add 5 new handler methods to SceneHandlers class:**

Add import at top: `from AbletonMCP_Remote_Script.handlers.tracks import COLOR_NAMES, COLOR_INDEX_TO_NAME`

1. **`_set_scene_color`** (`@command("set_scene_color", write=True)`):
   - Params: `scene_index` (int, default 0), `color` (str)
   - Validate scene_index range against `self._song.scenes`
   - Validate color exists in `COLOR_NAMES` dict, raise ValueError with sorted valid colors if not
   - Set `scene.color_index = COLOR_NAMES[color]`
   - Return: `{"scene_index": scene_index, "name": scene.name, "color": color}`

2. **`_get_scene_color`** (`@command("get_scene_color")`):
   - Params: `scene_index` (int, default 0)
   - Validate scene_index range
   - Get color name via `COLOR_INDEX_TO_NAME.get(scene.color_index, f"unknown_{scene.color_index}")`
   - Return: `{"scene_index": scene_index, "name": scene.name, "color": color_name, "color_index": scene.color_index}`

3. **`_set_scene_tempo`** (`@command("set_scene_tempo", write=True)`):
   - Params: `scene_index` (int, default 0), `tempo` (float, optional), `enabled` (bool, default True)
   - Validate scene_index range
   - If tempo is not None: set `scene.tempo = tempo`
   - Set `scene.tempo_enabled = enabled`
   - Return: `{"scene_index": scene_index, "name": scene.name, "tempo": scene.tempo, "tempo_enabled": scene.tempo_enabled, "note": "Scene tempo will override global tempo when this scene is fired" if scene.tempo_enabled else "Scene tempo disabled, using song tempo"}`

4. **`_set_scene_time_signature`** (`@command("set_scene_time_signature", write=True)`):
   - Params: `scene_index` (int, default 0), `numerator` (int, optional), `denominator` (int, optional), `enabled` (bool, default True)
   - Validate scene_index range
   - If numerator is not None: set `scene.time_signature_numerator = numerator`
   - If denominator is not None: set `scene.time_signature_denominator = denominator`
   - Set `scene.time_signature_enabled = enabled`
   - Return: `{"scene_index": scene_index, "name": scene.name, "numerator": scene.time_signature_numerator, "denominator": scene.time_signature_denominator, "time_signature_enabled": scene.time_signature_enabled, "note": "Scene time signature will override global when this scene is fired" if scene.time_signature_enabled else "Scene time signature disabled, using song time signature"}`

5. **`_fire_scene_as_selected`** (`@command("fire_scene_as_selected", write=True)`):
   - Params: `scene_index` (int, default 0), `force_legato` (bool, default False)
   - Validate scene_index range
   - Set `self._song.view.selected_scene = scene` (select the scene first)
   - Call `scene.fire_as_selected(force_legato=force_legato)`
   - Return: `{"fired": True, "index": scene_index, "name": scene.name, "force_legato": force_legato, "note": "Scene fired and selection advanced to next scene"}`

6. **`_get_scene_is_empty`** (`@command("get_scene_is_empty")`):
   - Params: `scene_index` (int, default 0)
   - Validate scene_index range
   - Return: `{"scene_index": scene_index, "name": scene.name, "is_empty": scene.is_empty}`

**mixer.py -- Add 3 new handler methods to MixerHandlers class:**

7. **`_set_crossfader`** (`@command("set_crossfader", write=True)`):
   - Params: `value` (float)
   - Access via `self._song.master_track.mixer_device.crossfader`
   - The crossfader is a DeviceParameter, set `crossfader.value = value`
   - Return: `{"crossfader": crossfader.value, "min": crossfader.min, "max": crossfader.max}`

8. **`_set_crossfade_assign`** (`@command("set_crossfade_assign", write=True)`):
   - Params: `track_index` (int, default 0), `track_type` (str, default "track"), `assign` (int)
   - Use `_resolve_track` to get the track
   - Validate assign is in {0, 1, 2}: 0=A, 1=none, 2=B. Raise ValueError if not.
   - Raise ValueError if track_type == "master" ("Master track does not have crossfade assignment")
   - Set `track.mixer_device.crossfade_assign = assign`
   - Map assign to label: `{0: "A", 1: "none", 2: "B"}`
   - Return: `{"crossfade_assign": track.mixer_device.crossfade_assign, "assign_label": label, "name": track.name, "type": track_type, "index": track_index}`

9. **`_get_panning_mode`** (`@command("get_panning_mode")`):
   - Params: `track_index` (int, default 0), `track_type` (str, default "track")
   - Use `_resolve_track` to get the track
   - Map mode to label: `{0: "Stereo", 1: "Split Stereo"}`
   - Return: `{"panning_mode": track.mixer_device.panning_mode, "mode_label": label, "name": track.name, "type": track_type}`
   - Add `"index": track_index` if track_type != "master"
