**Module-level helpers (before class):**

1. `_to_db(value: float) -> str` -- Convert normalized 0.0-1.0 to dB approximation string using `20 * math.log10(value)`. Return "-inf dB" for value <= 0, rounded to 1 decimal otherwise (e.g., "-6.0 dB"). Import `math` at top.

2. `_pan_label(value: float) -> str` -- Convert pan value to human-readable label. Return "center" for value == 0.0, "X% left" for negative, "X% right" for positive (percentage = abs(value) * 100, rounded to int). Return "100% left" / "100% right" for extremes.

**6 command handlers inside MixerHandlers class (all with `write=True`):**

1. `_set_track_volume(self, params)` -- @command("set_track_volume", write=True)
   - Extract: track_index (default 0), volume, track_type (default "track")
   - Validate volume is not None, then 0.0 <= volume <= 1.0. On out-of-range: read current value from `track.mixer_device.volume.value`, raise ValueError with current value in message
   - Set `track.mixer_device.volume.value = volume`
   - Read back actual: `track.mixer_device.volume.value`
   - Return dict: `{"volume": actual, "volume_db": _to_db(actual), "name": track.name, "type": track_type}` plus `"index": track_index` when track_type != "master"

2. `_set_track_pan(self, params)` -- @command("set_track_pan", write=True)
   - Extract: track_index (default 0), pan, track_type (default "track")
   - Validate pan is not None, then -1.0 <= pan <= 1.0. On out-of-range: read current from `track.mixer_device.panning.value`, raise ValueError with current value
   - Set `track.mixer_device.panning.value = pan`
   - Read back actual: `track.mixer_device.panning.value`
   - Return dict: `{"pan": actual, "pan_label": _pan_label(actual), "name": track.name, "type": track_type}` plus `"index": track_index` when track_type != "master"

3. `_set_track_mute(self, params)` -- @command("set_track_mute", write=True)
   - Extract: track_index (default 0), mute (bool), track_type (default "track")
   - Guard: if track_type == "master", raise ValueError("Master track cannot be muted")
   - Set `track.mute = mute`
   - Return dict: `{"mute": track.mute, "name": track.name, "type": track_type}` plus `"index": track_index` when track_type != "master"

4. `_set_track_solo(self, params)` -- @command("set_track_solo", write=True)
   - Extract: track_index (default 0), solo (bool), track_type (default "track"), exclusive (default False)
   - Guard: if track_type == "master", raise ValueError("Master track cannot be soloed")
   - If exclusive and solo: iterate `self._song.tracks` and `self._song.return_tracks`, set `t.solo = False` for any track where `t.solo` is True
   - Set `track.solo = solo`
   - Return dict: `{"solo": track.solo, "name": track.name, "type": track_type}` plus `"index": track_index` when track_type != "master"

5. `_set_track_arm(self, params)` -- @command("set_track_arm", write=True)
   - Extract: track_index (default 0), arm (bool), track_type (default "track")
   - Resolve track, then check `hasattr(track, 'can_be_armed') and track.can_be_armed`
   - If not armable: raise ValueError with explanation (e.g., "Track 'GroupName' (type) cannot be armed. Only MIDI and audio tracks support arming.")
   - Set `track.arm = arm`
   - Return dict: `{"arm": track.arm, "name": track.name, "type": track_type}` plus `"index": track_index` when track_type != "master"

6. `_set_send_level(self, params)` -- @command("set_send_level", write=True)
   - Extract: track_index (default 0), track_type (default "track"), return_index, level
   - Resolve track. If track_type == "master": raise ValueError("Master track does not have sends -- it receives the mix bus output directly")
   - Get sends = track.mixer_device.sends
   - Validate return_index: 0 <= return_index < len(sends). On out-of-range: raise IndexError with message including count of return tracks via len(self._song.return_tracks)
   - Validate level: 0.0 <= level <= 1.0. On out-of-range: read current from sends[return_index].value, raise ValueError with current value
   - Set `sends[return_index].value = level`
   - Read back actual: `sends[return_index].value`
   - Get return track name: `self._song.return_tracks[return_index].name`
   - Return dict: `{"send_level": actual, "send_db": _to_db(actual), "return_index": return_index, "return_name": return_name, "name": track.name, "type": track_type}` plus `"index": track_index` when track_type != "master"
