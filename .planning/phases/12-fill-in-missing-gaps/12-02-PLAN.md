Add these commands:

1. `@command("create_arrangement_midi_clip", write=True)` → `_create_arrangement_midi_clip(self, params)`:
   - Accepts: track_index (int), start_time (float, beats), length (float, beats), track_type (str, default "track")
   - Resolve track via `_resolve_track(self._song, track_type, track_index)`
   - Call `track.create_midi_clip(start_time, length)` (LOM arrangement API, NOT clip_slots)
   - Note: this creates in arrangement timeline, not session view
   - Returns `{"created": True, "track_name": track.name, "start_time": start_time, "length": length}`

2. `@command("create_arrangement_audio_clip", write=True)` → `_create_arrangement_audio_clip(self, params)`:
   - Accepts: track_index (int), file_path (str), position (float, beats), track_type (str, default "track")
   - Resolve track, call `track.create_audio_clip(file_path, position)`
   - Trust the file path -- let Ableton error if not found (per CONTEXT.md decision)
   - Returns `{"created": True, "track_name": track.name, "file_path": file_path, "position": position}`

3. `@command("get_arrangement_clips")` → `_get_arrangement_clips(self, params)`:
   - Accepts: track_index (int), track_type (str, default "track")
   - Resolve track, iterate `track.arrangement_clips`
   - For each clip: `{"name": clip.name, "start_time": clip.start_time, "end_time": clip.end_time, "length": clip.length, "is_audio_clip": clip.is_audio_clip, "color": COLOR_INDEX_TO_NAME.get(clip.color_index, f"unknown_{clip.color_index}")}`
   - Returns `{"track_name": track.name, "clips": [...]}`

4. `@command("duplicate_clip_to_arrangement", write=True)` → `_duplicate_clip_to_arrangement(self, params)`:
   - Accepts: track_index (int), clip_index (int), arrangement_time (float, beats)
   - Resolve clip_slot via `_resolve_clip_slot` (import from clips.py)
   - Validate clip_slot.has_clip
   - Get the clip object, then: `track.duplicate_clip_to_arrangement(clip_slot.clip, arrangement_time)`
   - Returns `{"duplicated": True, "clip_name": clip_slot.clip.name, "arrangement_time": arrangement_time}`
