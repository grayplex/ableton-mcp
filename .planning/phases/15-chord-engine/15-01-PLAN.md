    **Function 1: `build_chord(root, quality, octave=4)`** (per D-03, D-04, D-05):
    - Look up quality in `_QUALITY_MAP`; if not found, try passing raw quality to ChordSymbol as fallback; raise ValueError if ChordSymbol also fails
    - Build: `harmony.ChordSymbol(root + mapped_quality)`
    - Transpose to target octave: compute `(octave - chord_root_octave) * 12` semitones, transpose all pitches
    - Return dict: `{"root": str, "quality": str, "notes": [note_obj, ...]}` with notes ordered bottom to top

    **Function 2: `get_chord_inversions(root, quality, octave=4)`** (per D-07, D-11):
    - Build chord via `build_chord()` to get base pitches
    - Create music21 Chord object from the MIDI values
    - For n in range(len(pitches)): create inversion by rotating bottom n notes up an octave
    - Return list of dicts: `[{"inversion": 0, "bass_note": "C4", "notes": [...]}, ...]`

    **Function 3: `get_chord_voicings(root, quality, octave=4)`** (per D-08):
    - Build chord via `build_chord()` to get close-position MIDI values
    - Close: the default output
    - Open: drop every other note (index 1, 3, ...) from bottom by 1 octave, re-sort
    - Drop-2: take 2nd-from-top pitch, subtract 12, re-sort. Requires 3+ notes.
    - Drop-3: take 3rd-from-top pitch, subtract 12, re-sort. Requires 4+ notes; return `null` if fewer.
    - Rebuild note objects from resulting MIDI values using `_make_note_obj`
    - Return dict: `{"close": [...], "open": [...], "drop2": [...], "drop3": [...] | null}`

    **Function 4: `identify_chord(midi_pitches)`** (per D-09, D-10, D-11):
    - Create `chord.Chord(midi_pitches)`
    - Primary candidate: extract `root()`, `quality`, `pitchedCommonName`, `inversion()`, `bass()`
    - Generate alternative candidates: try each pitch in the set as potential root, build chord from intervals, check match
    - Score: root position = 1.0, inversions = 0.8 - 0.05*inversion_num, enharmonic reinterpretation = 0.6
    - Return top 3 candidates sorted by confidence: `[{"root": str, "quality": str, "name": str, "inversion": int, "bass_note": str, "confidence": float, "notes": [...]}]`

    **Function 5: `get_diatonic_chords(key_name, scale_type="major", octave=4)`** (per D-12, D-13, D-14):
    - Only accept "major" and "minor" for scale_type (per D-14); raise ValueError otherwise
    - Major key Roman numerals for triads: `["I", "ii", "iii", "IV", "V", "vi", "viio"]`
    - Minor key Roman numerals for triads: `["i", "iio", "III", "iv", "v", "VI", "VII"]`
    - Major 7ths: `["Imaj7", "ii7", "iii7", "IVmaj7", "V7", "vi7", "viio7"]`
    - Minor 7ths: `["i7", "iio7", "IIImaj7", "iv7", "v7", "VImaj7", "VII7"]`
    - Build each via `roman.RomanNumeral(numeral, key.Key(key_name, scale_type))`
    - Transpose to target octave
    - Return: `{"key": str, "scale": str, "triads": [chord_dict, ...], "sevenths": [chord_dict, ...]}`
    - Each chord_dict: `{"degree": int, "roman": str, "quality": str, "root": str, "notes": [note_obj, ...]}`
