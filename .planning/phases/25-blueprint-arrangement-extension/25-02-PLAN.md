```python
class _ArrangementEntryRequired(TypedDict):
    name: str
    bars: int

class ArrangementEntry(_ArrangementEntryRequired, total=False):
    energy: int              # 1-10 energy level
    roles: List[str]         # Active instrument roles in this section
    transition_in: str       # How to transition into this section
```

Rules (from CONTEXT.md decisions):
- D-04: Intro/first sections OMIT transition_in entirely (no key, no empty string)
- D-05: roles are a soft subset of the genre's instrumentation.roles
- D-06: Energy curve is genre-specific: intro ~2-3, buildups ramp, drops peak 8-10, breakdowns 3-5, outro 2-3
- D-07: roles lists only ACTIVE instruments (not silent ones)
- D-08: transition_in is a short actionable phrase (3-6 words)
